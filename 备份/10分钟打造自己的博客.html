##使用hexo搭建个人博客

###前提是已经具备github推送代码环境

###安装node.js 去官网直接下载，下一步就可以

###安装Hexo

1. $ npm install -g hexo （用git执行，可能会遇到万络问题）
2. $ git init /$ hexo init(安装完毕后行新建一个文件夹，到里面打开git初始化git和hexo )
3. $ hexo g/$ hexo s(分别执行代码可以试试本地的博客是否搭建成功 网址:localhost:4000)

###更换主题

1. $ hexo clean/$ hexo g/$ hexo s (每次更换主题都要重新clean一下)
2. $cd themes/$ git clone https://github.com/iissnan/hexo-theme-next.git
3. 修改目录下的config.yml配置文件中的theme属性，将其设置为hexo-theme-next
4. $ hexo g/$ hexo s（查看themes是否成功）

###上传到github

1. 首先你必须在gitbub上新建一个仓库保证能推送上去（特别注意命名 yourgithubname.github.io）
2. 修改目录下的config.yml配置文件
  • deploy:
  • type: git
  • repository: https://github.com/yourgithubname /yourgithubname .github.io.git (地址是复制的)
  • branch: master
    ###上传
    1. $ hexo g/$ hexo d
    2. $ npm install hexo-deployer-git –save(如果在执行 hexo deploy 后,出现 error deployer not found:github 的错误)
